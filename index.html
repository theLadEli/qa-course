Documentation:

Above I have created a form with multiple form fields. On form submit the formmSubmission(event) function is called, along with an onclick event that's added to the Submit button, this calls the storeName function. The storeName uses DOM to get the value of the Name field and set it to a variable. Next, then the formSubmission function removes the default form submit settings, displays an alert with some text and resets the form.
