Here is the JavaScript used for the form submit actions:

        
          // The below had to be declared outside of the formSubmission function for it to work
          var yourName;
          var yourEmail;
          var yourSubject;
          var yourMessage;
          
          //A function called when the form is submitted
          function formSubmission(event) {
              //Prevents the default form submission actions
              event.preventDefault();
          
              //Sets the variable values to the form field values
              yourName = document.getElementById("name").value;
              yourEmail = document.getElementById("email").value;
              yourSubject = document.getElementById("subject").value;
              yourMessage = document.getElementById("message").value;
          
              //Updates the submission toast with the values
              document.getElementById('toast-submission-details').innerHTML = (`
                <p style="font-size: 14px;">
                  <b>Name:</b> ${yourName}
                  <br><b>Email:</b> ${yourEmail}
                  <br><b>Subject:</b> ${yourSubject}
                  <br><b>Message:</b> ${yourMessage}
                </p>`
              )
          
              //Unhides the toast
              document.getElementById("form-submit-toast").style.display = "inline";
          
              //Resets the form
              document.getElementById("myForm").reset();
          }
          
          //Exit toast
          function closeToast(){
              document.getElementById('form-submit-toast').style.display = 'none';
          }

When the form is submitted it calls the formSubmission() function. This function does a few things:

  1. It first stops the default form submission actions from happening
  2. Gets the value of the form fields and saves them to variables
  3. Sets the innerHTML of the Form Successfully Submitted popup text to a concatenated version containing all the variables I just created. The popup is still hidden at this point.
  4. It unhides the popup by setting its style.display value to inline
  5. Clears the form

Note: This form does not store entires, this can be accomplished using PHP

Conditional Logic

By entering the number 5 in the below field, you will change the status light from red to green!

Status: 🔴

Status: 🟢

Documentation:

The conditional logic form works by having two elements, one with the green circle emoji, the other with a red circle emoji. There are also two classes that target each emoji and set them to hidden. By default, when you load onto the page the red emoji has no class but the green emoji has the class that sets it to hidden.

In JavaScript I created an if else statement that add the red emoji class if the number is five, this sets the red emoji to hidden. It then removes the green emoji's hidden class.

The code looks like so: 

      
        var conditionalNumber = document.getElementById("conditional-number").value;
        console.log(conditionalNumber)
    
        if (conditionalNumber == 5) {
            console.log("submitted 5");
            document.getElementById("conditional-red").classList.add('conditional-logic-red');
            document.getElementById("conditional-green").classList.remove('conditional-logic-green');
        } else if (conditionalNumber > 5 || conditionalNumber < 5) {
            document.getElementById("conditional-red").classList.remove('conditional-logic-red');
            document.getElementById("conditional-green").classList.add('conditional-logic-green');
        }

The JavaScript events section has three elements in it:

  1. A <p> element, it has an id of js-events-p
  2. A Show button with an onclick event to call the showButton() function
  3. A Hide button with an onclick event to call the hideButton() function

The showButton() function is a simple JS function that targets the js-events-p class and sets its display to block. The hideButton() function almost the same, except it sets the display to none instead of block.
