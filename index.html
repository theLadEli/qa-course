Documentation:

The conditional logic form works by having two elements, one with the green circle emoji, the other with a red circle emoji. There are also two classes that target each emoji and set them to hidden. By default, when you load onto the page the red emoji has no class but the green emoji has the class that sets it to hidden.

In JavaScript I created an if else statement that add the red emoji class if the number is five, this sets the red emoji to hidden. It then removes the green emoji's hidden class.

The code looks like so: 

      
        var conditionalNumber = document.getElementById("conditional-number").value;
        console.log(conditionalNumber)
    
        if (conditionalNumber == 5) {
            console.log("submitted 5");
            document.getElementById("conditional-red").classList.add('conditional-logic-red');
            document.getElementById("conditional-green").classList.remove('conditional-logic-green');
        } else if (conditionalNumber > 5 || conditionalNumber < 5) {
            document.getElementById("conditional-red").classList.remove('conditional-logic-red');
            document.getElementById("conditional-green").classList.add('conditional-logic-green');
        }
